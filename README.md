## (1) 요약  
이화이언의 수습 운영진 모집 사이트   
지원서를 다운받고 업로드 할 수 있으며, 운영진 합격 및 불합격 결과를 알 수 있다.  
이외에도, 이화이언의 대략적인 설명을 볼 수 있는 사이트이다.
